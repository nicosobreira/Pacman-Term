# Pacman Term

## Build

### Linux

#### Dependencies

On Ubuntu/Debian
``` bash
sudo apt install cmake
```

Enter the following command on the terminal
``` bash
mkdir build && cd build
cmake ../
make
```

Then run `./pacman`.
